## 📌 Project Summary
This project investigates the major **factors affecting health insurance claim costs** using a structured dataset and advanced Excel analytics techniques.

The core objective is to understand how personal and lifestyle variables influence **medical insurance charges**, helping stakeholders make informed decisions on **pricing, risk assessment, and claims forecasting**.

### Variables Used
- **Age**
- **BMI**
- **Smoking Status**
- **Number of Children**
- **Sex**
- **Region**
- **Insurance Charges**

---

## 🎯 Project Objectives
- To identify key factors influencing health insurance charges
- To analyze relationships between variables (e.g., age vs charges)
- To determine which factors have the strongest impact on claims
- To build simple predictive insights using Excel tool

## 📈 Key Insights

* **Age has a positive effect** on insurance charges.
* **Smoking status is the strongest driver** of high claims.
* **Children show a weak but positive relationship** with cost.
* BMI contributes moderately depending on health risk.
* Regression confirms the **statistical significance of major predictors**.
